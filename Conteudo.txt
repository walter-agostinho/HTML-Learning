TAGS
    <head> = cabeçalho
    <body> = corpo do texto
    <p> = parágrafo
    <h1> até <h6> = niveis de titulo de seção. h1 sendo o mais alto
    <br> = pula a linha
    
    <a> = âncora, usado para link
    PARÂMETROS
        href = local do link
        target = onde abrirá o link
        rel = relacionado a que
        media = para qual mídia

    <abbr> = abreviação. Deixando mouse mostra o titulo
    PARÂMETROS
        title = titulo que será mostrado ao deixar o mouse

    <address> = informações de contato do autor

    <map> = utilizado em conjunto com area. Pai de area. Serve para mapear area em uma imagem
    PARÂMETROS
        name

    <area>
    PARÂMETROS
        alt = nome
        shape = forma, ex poligono
        href = referencia ao clickar
        coords = coordenadas
    
    <article> = área ou seção idependente do documento

    <aside> = representa uma seção da pagina tangencialmente relacionado ao conteudo do elemento aside

    <audio>
        PARÂMETROS
        autoplay = ja começa com audio
        controls = exibe os controles do audio
        muted = começa mudo
        loop = reproduz audio novamente

    <b> = texto em negrito

    <base> = caminho base para os arquivos

    <bdo> = inverte o texto
        PARÂMETROS
        dir = ltr = inverte o texto

    <blockquote> = citação
        PARÂMETROS
        cite = link para conteudo externo

    <cite> = referencia a obra e os autores

    <br> = quebra de linha

    <button> = botão para ações
        PARÂMETROS
        autofocus = foca quando pagina carregada
        type = tipo do botão

    <canvas> = renderizar imagens, usado para jogos

    <code> = para destacar trecho de codigo

    <pre> = bloco de texto pre formatado. Exibe exatamente como esta o texto

    <del> = representa um conteudo excluido ou marcado para exclusao (texto fica riscado)
        PARÂMETROS
        cite = referência
        datetime = data da exclusão

    <ins> = representa uma inserção (texto fica sublinhado)
        PARÂMETROS
        cite = referência
        datetime = data da inserção
    
    <details> = específica detalhes adicionais que podem ser abertos ou fechados

    <summary> = específica o cabeçalho de details

    <dfn> = definição. Quando tiver um termo com definição
        PARÂMETROS
        title = exibe quando deixa o mouse

    <dialog> = caixa de dialogo
    
    <div> = container generico, organização dos elementos

    <dl> = lista de nome valor

    <dt> = chave do <dl>

    <dd> = valor do <dt>

    <em> = dar enfase

    <embed> = geral para musica, video etc

    <figure> marcar uma imagem dentro de um documento
    <figcaption> legenda para a marcação

    <footer> tag de seção para rodapé

Informação
    display block = não permite elemento na mesma linha
    display inline = permite outro elemento na mesma linha
